🔥 Win10 安装 U 盘制作终极命令（100% 解决 0 字节/找不到文件）

🔧最终解决命令（核心修复）

确认：当前目录存在的镜像名是 Win10_22H2_x64.iso

# 1. 卸载 U 盘分区（避免写入冲突）
sudo umount /dev/sdb*

# 2. 写入 U 盘（使用真实存在的文件名）
sudo dd if=Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress
sync

📁文件确认命令（防止手输错）

如果上面还报错，先执行这个查看名字

ls /home/kali/Downloads | grep Win10

📖完整安装教程（删掉 Kali）

  • 1. 写入完成：命令执行完，看到 `1+0 records in` 才算成功。
  • 2. 重启：拔掉 U 盘重新插上，或者直接重启电脑。
  • 3. 启动项：惠普笔记本开机狂按 F9，选择 USB Storage 启动。
  • 4. 关键一步：进入安装界面，选择 自定义：仅安装 Windows。
  • 5. 格式化：选中所有 Kali Linux 分区，点击 删除，直到变成 未分配空间。
  • 6. 安装：选中未分配空间，点击下一步，静静等待安装完成。

💡避坑指南

  • 绝对不要把 `of=/dev/sdb` 写成 `of=/dev/sda`，那会格式化你的电脑硬盘！
  • 绝对不要用长文件名，就用 `Win10_22H2_x64.iso`。
  • 情绪的事先放放，等你装完 Windows，删掉所有 Kali，生活也是全新的。
