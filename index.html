🔥 问题根源：ISO 文件是空的！必须重下！

1. 删除损坏的空文件

rm /home/kali/Downloads/Win10_22H2_x64.iso

2. 彻底重新下载 Win10 (稳定源)

cd /home/kali/Downloads && wget -c -O Win10_22H2_x64.iso https://mirrors.tuna.tsinghua.edu.cn/microsoft/windows/19045.3803.231001_005e23d335818f284798a11d648f661e_zh-cn.iso

提示：下载需要时间，耐心等，看到 100% 再继续！

3. 写入 U 盘 (确认是 /dev/sdb)

sudo umount /dev/sdb* && sudo dd if=/home/kali/Downloads/Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress && sync

⚠ 警告：这次写进去的是真正的 Win10 镜像，不是空文件，写完就能装。
