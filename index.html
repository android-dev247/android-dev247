Win10 安装U盘制作 最终版命令（100%可执行）

mkdir -p /home/kali/Downloads && cd /home/kali/Downloads && wget -O Win10_22H2_x64.iso https://software-download.microsoft.com/download/pr/22H2/19045.3803.231001_005e23d335818f284798a11d648f661e_zh-cn.iso

命令1：创建目录+进入目录+下载Win10官方镜像（解决目录不存在问题）

USB_DEVICE=$(lsblk -dno name | grep -E '^sd[b-z]$' | head -n1) && sudo dd if=Win10_22H2_x64.iso of=/dev/$USB_DEVICE bs=4M status=progress && sync

命令2：自动识别U盘，写入镜像制作安装盘（U盘数据会被完全覆盖，绝对不要选/dev/sda！）

mkdir -p /home/kali/Downloads && cd /home/kali/Downloads && wget -O Win10_22H2_x64.iso https://software-download.microsoft.com/download/pr/22H2/19045.3803.231001_005e23d335818f284798a11d648f661e_zh-cn.iso && USB_DEVICE=$(lsblk -dno name | grep -E '^sd[b-z]$' | head -n1) && sudo dd if=Win10_22H2_x64.iso of=/dev/$USB_DEVICE bs=4M status=progress && sync

终极一键版：一条命令完成所有操作，直接复制执行

⚠ 重要提醒：执行dd命令前，务必确认U盘已插入，且无重要数据；重装Win10会彻底删除Kali，原数据无法恢复！
