Win10 U盘写入 最终零错误命令（解决找不到文件）

ls /home/kali/Downloads | grep Win10

命令1：查看真实镜像文件名（复制显示的完整名字）

sudo dd if=Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress && sync

命令2：用真实镜像名写入U盘（U盘数据会被完全覆盖，绝对不要选/dev/sda！）

cd /home/kali/Downloads && ls | grep Win10 && sudo dd if=Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress && sync

终极一键版：一条命令完成查看+写入U盘

⚠ 重要提醒：执行dd命令前，务必确认U盘已插入；重装Win10会彻底删除Kali，原数据无法恢复！
