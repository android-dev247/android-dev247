⚠️ 解决 400 错误 - 重装 Windows 终极命令

第一步：安装下载依赖（必执行）
apt update && apt install -y wget curl
第二步：下载 Windows 镜像（稳定链接）
wget -O Win10_22H2_x64.iso https://massgrave.dev/r/Win10_22H2_x64_CN.html
第三步：写入 U 盘（确认 U 盘设备名）
# 查看 U 盘设备名 lsblk -dno name | grep -E '^sd[b-z]$' # 写入镜像（替换 /dev/sdb 为实际 U 盘设备） dd if=Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress sync
关键说明：
  • 执行 `dd` 命令前，务必确认 U 盘设备名（不要写成 /dev/sda）
  • 重装过程中，保持网络稳定，避免中断
  • 写入完成后，重启电脑从 U 盘启动安装
