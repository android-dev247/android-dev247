🔥 Win10 重装终极教程（彻底解决 404/下载失败/链接失效）

⚠️核心原因：wget 直链被封

微软和清华源的直链在 Kali Linux 下访问会被防火墙拦截/链接过期，导致 404 Not Found。

解决方案：放弃命令行下载，直接用官方工具下载，或者手动复制备用链接。

⚙️一键制作 U 盘命令（下载完镜像后执行）

假设你下载好的 ISO 文件名是 Win10_22H2_x64.iso，放在 /home/kali/Downloads

# 1. 进入下载目录
cd /home/kali/Downloads

# 2. 查看 U 盘设备名 (绝对不要选 /dev/sda)
lsblk -dno name | grep -E '^sd[b-z]$'

# 3. 写入镜像 (替换 /dev/sdb 为实际 U 盘名)
sudo dd if=Win10_22H2_x64.iso of=/dev/sdb bs=4M status=progress
sync

📖完整安装教程（从 U 盘启动到装完）

  • 第一步：插入 U 盘重启
    惠普笔记本开机连续按 F9，选择 USB 启动项。
  • 第二步：选择语言
    进入安装界面，选择中文（简体）、中文（繁体）、美国，点击现在安装。
  • 第三步：输入密钥（跳过）
    点击 我没有产品密钥（装完激活）。
  • 第四步：选择版本
    选择 Windows 10 专业版（或你需要的版本），点击下一步。
  • 第五步：选择安装类型（关键）
    选择 自定义：仅安装 Windows（高级）。
  • 第六步：删除所有 Kali 分区
    选中每个分区（显示为 sda1, sda2...），点击 删除，直到全部变成 未分配空间。
  • 第七步：安装系统
    选中 未分配空间，点击 下一步。电脑会自动安装，期间会重启 3-4 次，千万不要拔 U 盘。
  • 第八步：设置账户
    等待安装完成，设置用户名和密码，进入 Windows 桌面。

❌避坑指南

  • 绝对不要把 Windows 装在 U 盘里，必须装在 /dev/sda（电脑硬盘）。
  • 绝对不要用 `dd` 命令写 `/dev/sda`，会把 Kali 彻底删光。
  • 如果 U 盘启动不了，检查 U 盘是否制作完整，或者换个 U 盘。
  • 装完 Windows 后，去微软官网下载驱动，确保显卡/声卡正常。
