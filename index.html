Win10 U盘写入 最终修复命令（解决0字节写入）

sudo dd if=Win10_22H2_Chinese_Simplified_x64v1.iso of=/dev/sdb bs=4M status=progress && sync

命令1：用正确镜像名写入U盘（U盘数据会被完全覆盖，绝对不要选/dev/sda！）

# 1. 进入下载目录
cd /home/kali/Downloads

# 2. 用正确镜像名写入U盘
sudo dd if=Win10_22H2_Chinese_Simplified_x64v1.iso of=/dev/sdb bs=4M status=progress
sync

分步执行版：分两条命令，方便排查问题

cd /home/kali/Downloads && sudo dd if=Win10_22H2_Chinese_Simplified_x64v1.iso of=/dev/sdb bs=4M status=progress && sync

终极一键版：一条命令完成所有操作，直接复制执行

⚠ 重要提醒：执行dd命令前，务必确认U盘已插入；重装Win10会彻底删除Kali，原数据无法恢复！
